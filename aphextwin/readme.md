# 🎵 Site sur Aphex Twin

Site web réalisé dans le cadre d'un projet scolaire, sur un artiste au choix.

## 🎯 Objectif

Créer un site simple présentant la démarche artistique d'un artiste.
J'ai choisi **Aphex Twin** (Richard D. James), figure majeure de la musique électronique expérimentale.

## 📄 Contenu

Le site aborde :
- Le parcours et l'univers d'Aphex Twin
- Sa démarche artistique et ses influences
- Son impact sur la musique électronique

## 🛠️ Technologies

- HTML
- CSS
